# to-bus
Technologie obiektowe 2

Documentation on wiki page.

Project use Play framework and PostgreSQL database.

## to run 

```bash
sbt run
```
Go to `localhost:9000` in browser

tested on firefox, there were problems on Opera

## to test

```bash
sbt test
```

## to build package

```bash
sbt package
```

## database 

PostgreSQL

## compatibility

tested on firefox, there were problems on Opera
